Fundamentos React e JSX

Component

Um componente é uma função que retorna elementos da interface. É a unidade básica de construção de aplicações React. 


function Welcome() {
  return <h1>Olá Mundo</h1>;
}

Props

Props são propriedades usadas para passar dados de um componente pai para um componente filho. 


function Welcome(props) {
  return <h1>Olá, {props.nome}</h1>;
}

<Welcome nome="Vitor" />

useState

Hook que permite criar e atualizar estados em componentes funcionais. 


import { useState } from "react";

function Contador() {
  const [numero, setNumero] = useState(0);

  return (
    <button onClick={() => setNumero(numero + 1)}>
      {numero}
    </button>
  );
}

JSX

Sintaxe semelhante ao HTML usada para descrever a interface no React. 

const elemento = <h1>Olá, mundo!</h1>

Fragment

Uma sintaxe que permite agrupar uma lista de filhos sem adicionar nós extras ao DOM (HTML gerado). 


<>
  <h1>Título</h1>
  <p>Texto</p>
</>

className

Atributo utilizado para aplicar classes CSS em elementos React. 


<h1 className="titulo">Olá</h1>

onClick

Evento executado quando o usuário clica em um elemento. 


<button onClick={() => alert("Clicou!")}>
  Clique
</button>

onChange

Evento acionado quando o valor de um campo é alterado. 


<input onChange={(e) => console.log(e.target.value)} />

value

Define ou acessa o valor de um campo de formulário. 


<input value="React" />

placeholder

Texto exibido dentro de um campo enquanto ele está vazio. 


<input placeholder="Digite seu nome" />

Manipulação de Inputs e Eventos

event

É o objeto gerado automaticamente pelo navegador quando um evento ocorre (como um clique ou digitação), contendo informações detalhadas sobre o que aconteceu. 


function gerenciarClique(event) { console.log(event); }

e.target.value

Uma propriedade do objeto de evento (e) usada para acessar o valor atual de um elemento de entrada (input) no momento em que o evento é disparado. 


<input onChange={(e) => console.log(e.target.value)} />

preventDefault()

Um método do objeto de evento utilizado para cancelar o comportamento padrão do navegador, muito comum para evitar que a página recarregue ao enviar um formulário. 


function enviar(e) { e.preventDefault(); }

input type="text"

Um elemento de formulário padrão usado para criar um campo de texto de linha única, onde o usuário pode digitar dados. 


<input type="text" name="usuario" />

button

Elemento de interface que define um botão clicável, utilizado para acionar ações ou submeter dados dentro do componente. 


<button onClick={handleClick}>Enviar</button>

textarea

Um elemento de entrada de dados que permite ao usuário digitar blocos maiores de texto em múltiplas linhas. 


<textarea placeholder="Digite sua mensagem"></textarea>

select

Elemento utilizado para criar uma lista suspensa (menu do tipo dropdown), permitindo que o usuário escolha uma ou mais opções pré-definidas. 


<select>
  <option value="opcao1">Opção 1</option>
  <option value="opcao2">Opção 2</option>
</select>

checked

Um atributo booleano utilizado em inputs do tipo checkbox ou radio para definir ou ler se o elemento está marcado ou não. 


<input type="checkbox" checked={maarcado}/>

onSubmit

Um manipulador de eventos aplicado ao elemento de formulário (

) que é disparado quando o usuário tenta enviar os dados (seja clicando em um botão ou apertando Enter). 


<form onSubmit={lidarComEnvio}>
  <button type="submit">Enviar</button>
</form>

disabled

Um atributo booleano que, quando verdadeiro, desativa a interatividade de um elemento (como inputs ou botões), impedindo ações do usuário. 


<button disabled>Enviar</button>

Arrays e Renderização

map()

Método que cria um novo array com os resultados da aplicação de uma função a cada elemento do array original. 


const numeros = [1, 2, 3];
const quadrados = numeros.map(num => num * num);
console.log(quadrados); // [1, 4, 9]

filter()

Método que cria um novo array com os elementos que atendem a uma condição específica. 


const numeros = [1, 2, 3, 4, 5];
const pares = numeros.filter(num => num % 2 === 0);
console.log(pares); // [2, 4]

find()

Método que retorna o primeiro elemento que atende a uma condição específica. 


const numeros = [1, 2, 3, 4, 5];
const primeiroPar = numeros.find(num => num % 2 === 0);
console.log(primeiroPar); // 2

push()

Método que adiciona um ou mais elementos ao final de um array. 


const numeros = [1, 2, 3];
numeros.push(4);
console.log(numeros); // [1, 2, 3, 4]

spread operator (...)

Permite expandir os elementos de um array em outro array ou em uma lista de argumentos. 


const numeros = [1, 2, 3];
const outrosNumeros = [...numeros, 4, 5];
console.log(outrosNumeros); // [1, 2, 3, 4, 5]

length

Propriedade que retorna o número de elementos em um array. 


const numeros = [1, 2, 3];
console.log(numeros.length); // 3

index

Propriedade que retorna o índice do elemento atual no array. 


const numeros = [10, 20, 30];
numeros.forEach((num, index) => {
  console.log(`Número: ${num}, Índice: ${index}`);
});

key

Propriedade utilizada pelo React para identificar quais elementos foram alterados, adicionados ou removidos. 


const itens = ['Maçã', 'Banana', 'Laranja'];
const lista = itens.map((item, index) => (
  <li key={index}>{item}</li>
));

includes()

Método que verifica se um elemento específico está presente em um array. 


const frutas = ['Maçã', 'Banana', 'Laranja'];
console.log(frutas.includes('Banana')); // true

sort()

Método que ordena os elementos de um array. 


const numeros = [3, 1, 4, 1, 5];
numeros.sort((a, b) => a - b);
console.log(numeros); // [1, 1, 3, 4, 5]

Strings e Dados

toUpperCase()

Método que converte uma string para maiúsculas. 


const texto = "olá, mundo!";
console.log(texto.toUpperCase()); // "OLÁ, MUNDO!"

toLowerCase()

Método que converte uma string para minúsculas. 


const texto = "OLÁ, MUNDO!";
console.log(texto.toLowerCase()); // "olá, mundo!"

trim()

Método que remove espaços em branco do início e do final de uma string. 


const texto = "  olá, mundo!  ";
console.log(texto.trim()); // "olá, mundo!"

split()

Método que divide uma string em um array de substrings. 


const texto = "olá, mundo!";
console.log(texto.split(",")); // ["olá", " mundo!"]

replace()

Método que substitui uma substring por outra. 


const texto = "olá, mundo!";
console.log(texto.replace("mundo", "universo")); // "olá, universo!"

template literals

Sintaxe que permite criar strings com interpolação de variáveis. 


const nome = "João";
const idade = 30;
console.log(`Olá, meu nome é ${nome} e eu tenho ${idade} anos.`); // "Olá, meu nome é João e eu tenho 30 anos."

parseInt()

Função que converte uma string em um número inteiro. 


const numStr = "123";
console.log(parseInt(numStr)); // 123

parseFloat()

Função que converte uma string em um número de ponto flutuante. 


const numStr = "123.45";
console.log(parseFloat(numStr)); // 123.45

Number()

Função que converte uma string em um número. 


const numStr = "123";
console.log(Number(numStr)); // 123

Boolean()

Função que converte uma valor em um booleano. 


console.log(Boolean("true")); // true
console.log(Boolean("false")); // true

Condicionais e Controle

if / else

Estrutura de controle que executa um bloco de código se uma condição for verdadeira. 


const idade = 18;
if (idade >= 18) {
  console.log("Você é maior de idade.");
} else {
  console.log("Você é menor de idade.");
}

Operador ternário ? :

Permite renderização condicional de forma mais concisa. 


const idade = 18;
const mensagem = idade >= 18 ? "Você é maior de idade." : "Você é menor de idade.";
console.log(mensagem);

&& no JSX

Permite renderização condicional de forma mais concisa. 


const idade = 18;
idade >= 18 && console.log("Você é maior de idade.");

return

Palavra-chave que encerra a execução de uma função e retorna um valor. 


function somar(a, b) {
  return a + b;
}

function

Palavra-chave que define uma função. 


function saudar(nome) {
  return `Olá, ${nome}!`;
}

Arrow Function =>

Sintaxe mais concisa para definir funções. 


const saudar = (nome) => `Olá, ${nome}!`;

const

Palavra-chave que define uma variável constante. 


const PI = 3.14;
PI = 3.14159; // Erro: não é possível reatribuir um valor a uma variável constante

let

Palavra-chave que define uma variável de escopo de bloco. 


let contador = 0;
contador++;
console.log(contador); // 1

export default

Palavra-chave que exporta uma função, classe ou variável como o valor padrão de um módulo. 


const saudar = (nome) => `Olá, ${nome}!`;
export default saudar;
